Необходимо доработать сайт в соответствии со следующим списком страниц (для примера см. скриншоты):

1. /home - главная страница, содержит список сообщений.
	Для каждого сообщения отображается дата создания, автор, тема сообщения, содержимое и количество комментариев.
	Присутствуют кнопки для добавления нового сообщения и перехода к просмотру существующего сообщения.

2. /message/add - страница добавления нового сообщения.
	Содержит поля для ввода имени автора, темы и содержимого сообщения, а также кнопки для сохранения сообщения и возврата на главную страницу.

3. /message/view/:messageID - страница для просмотра отдельного сообщения.
	Содержит полную информацию о сообщении, в том числе все комментарии и кнопки:
	а) возврат на главную страницу
	б) переход на страницу редактирования сообщения
	в) переход на страницу удаления сообщения
	г) переход на страницу, где осуществляется добавление оценки для сообщения
	д) переходы на страницу просмотра каждого комментария
	е) (опционально) переходы на страницу редактирования каждого комментария
	Также может включать в себя интерфейс добавления нового комментария.

4. /message/edit/:messageID - страница редактирования сообщения.
	Содержит поля для редактирования имени автора, темы и содержимого сообщения, а также кнопку для возврата к странице просмотра сообщения
	и кнопку для сохранения изменений. После сохранения изменений производится переход на страницу просмотра сообщения.

5. /message/vote/:messageID - страница добавления рейтинга для сообщения.
	Содержит информацию о сообщении, кнопки для выбора желаемого рейтинга, а также кнопку для возврата на страницу просмотра сообщения и
	кнопку для добавления оценки (после сохранения оценки производится переход на страницу просмотра сообщения).

6. /message/delete/:messageID - страница удаления сообщения.
	Содержит информацию об удаляемом сообщении, а также кнопку для возврата к просмотру сообщения и кнопку для окончательного удаления сообщения.
	После удаления сообщения производится переход на главную страницу.

7. /comment/view/:commentID - страница просмотра комментария.
	Содержит информацию о комментарии, кнопку для возврата к странице сообщения и кнопку для перехода на страницу редактирования сообщения.

8. /comment/edit/:commentID - страница редактирования комментария.
	Содержит информацию о сообщении, поля для изменения комментария (имя автора и содержимое комментария), а также кнопку для возврата
	на страницу просмотра комментария, и кнопку для сохранения изменений.
	После сохранения изменений производится переход на страницу просмотра комментария.
